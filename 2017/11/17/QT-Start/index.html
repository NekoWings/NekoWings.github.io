QT Start

发表于 2017-11-17 | 分类于 Python入门 | |

我还是回来学QT了。
半年前本来是要试着用QT做大一下程序设计基础的课设。最后拖到ddl因为上手太晚而放弃了它。但一直有一个制作GUI的想法.暂时没有时间和环境上手C#，或者Java。现在刚好趁着有项目可以做，学习一下QT和OpenCV。

QT的自带IDE还是不错的，只是某些地方一直有一些小bug，不知道是个人设置问题还是。每个项目都必须手动修改debug, release和profile目录才能正常编译运行。
运行了几个自带的example，还是灰常有意思的。

Calendar

作为一只猫，一定就要有搞事情的本性！

于是我想用我喜爱的Visual Studio去运行QT库。
当然，在VS2017(2015同) 工具->拓展和更新 中是有官方提供的插件的，顺利下载安装后会在上面一行菜单栏中出现Qt VS Tools一项。
然后选择Qt Options，“顺利”设置下去就行……
